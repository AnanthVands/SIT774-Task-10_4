The Parallax Scrolling Effect

  • The parallax scrolling effect was done using CSS.
  • A div element with the class "parallax" is created to contain the background image.
  • The background image is set using the CSS background-image property.
  • To create the parallax effect, the background-attachment property is set to "fixed".
  • The background-size is set to "cover" to cover the entire div area.

Tailored Bootstrap for Navigation, Cards, and Buttons

  • The navigation bar was created using the nav and navbar classes from Bootstrap.
  • The links in the navigation bar are styled with the nav-link class.
  • The custom color and hover effect was done using a custom-navbar and nav-link:hover property.
  • The cards are styled using Bootstrap's card class with more formatting in the CSS file.
  • The button is styled using Bootstrap's btn class.
  • Custom styles used instead of Bootstrap's default colors in the CSS.
  • Imported the "Urbanist" font from Google Fonts and applied using the CSS font-family property.

JSON Data Dynamically Loaded into Cards

  • JSON data about books including titles, authors, and ISBNs is stored in the server.
  • In the server, a route is defined to handle GET requests to the endpoint /books.
  • When the webpage is loaded, an event listener is added to the button
  • When the button is clicked, a GET request is made to the server endpoint /books to retrieve the JSON data.
  • The data is then converted to a JavaScript object using the .json() method.
  • For each book in the data, a new div element as a card class is created dynamically.
  • Inside each card the book's title, author, and ISBN are inserted.

Footer and Date Update Function

  • The footer is created using a <footer> element.
  • A JavaScript function updateFooterDate is used to update the footer with the current date.
  • The function retrieves the current date using new Date()
  • The formatted date is then inserted into a <span> element within the footer.
  • The DOMContentLoaded event is used to ensure the updateFooterDate function is executed once webpage had completely loaded and parsed.
